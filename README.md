# Chat

This chat is a simple javascript project

####Configuration
git clone this project into your VirtualHost directory

exemples:
- wamp/www/ for wamp
- cd mamp/apps/myApp/ &&  git clone https://github.com/yfuks/Chat.git htdocs --for mamp--


####Intall
`$> npm install`

####Start the server
`$> npm start`
